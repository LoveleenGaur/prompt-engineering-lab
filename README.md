# SHARP Prompt Evaluator

Evaluate, score, and improve prompts using the SHARP framework.

## Tech stack
- Streamlit
- Groq API
- SHARP Framework engine

## Setup
1. Install dependencies:
   `pip install -r requirements_groq.txt`
2. Run the app:
   `streamlit run app_groq.py`
3. Create a Groq API key at `https://console.groq.com/keys`
4. Paste the key into the sidebar

## Notes
- Recommended model in this app: `llama-3.3-70b-versatile`
- If Groq rate limits you, wait briefly and try again
